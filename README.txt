# INTERNET SPEED TEST

.requisitos
.python3
.modulo speedtest-cli (se puede instalar mediante pip: "pip install speedtest-cli)


.Este proyecto utiliza  el modulo speedtest-cli, que se conecta aun servidor de speedtest.net para 
obtener la velocidad de INTERNET

.El resultado puede variar dependiendo de su ubicacion geografica y de la carga en el servidor de speedtest

FUTURAS IMPLEMENTACIONES

.Agregar la posibilidad de elegir entre diferentes servidores de speedtest.net
.Guardar resultados en un archivo para un seguimiento de la velocidad de internet a lo largo del tiempo
